What's inside the bottle?

Wine OS is a small Linux project focused on being fun first: slightly crooked panels, soft wobble animations and bright neon accents. Underneath all that, it's still a normal Linux system with packages, updates and a real desktop you can live in.

Two spins GNOME for a clean, focused setup. KDE if you like tweaking every pixel. Same Wine OS sauce on top.
Pre‑mixed apps Browser, editor, terminal toys and theming stuff ready on first boot.
Real Linux core Package manager, repos, and all the boring reliable stuff you actually need.

Wine OS does not contain alcohol or promote drinking. Only your desktop looks drunk.

Install notes & requirements

  • CPU: 64‑bit x86_64
  • RAM: 4 GB min, 8 GB recommended
  • Disk: 20 GB free
  • Boot: USB stick ≥ 4 GB
  • Mode: UEFI or legacy (depending how you built it)

Flash the ISO with Ventoy, balenaEtcher, dd or whatever you like, boot from USB and run the installer. This is an early build – expect bugs, broken stuff and maybe a digital hangover.
